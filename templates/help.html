📊 {% if language=='ta' %}ஜாதக கணக்கீடு{% else %}Birth Chart Calculation{% endif %}

{% if language=='ta' %} லக்னம் என்பது நீங்கள் பிறந்த நேரத்தில் கிழக்கு எல்லையில் உதித்த ராசிக்கான பண்பு. இது உங்கள் உடல், ஆளுமை மற்றும் வாழ்க்கை திசையை குறிக்கிறது. சரியான நேரமும் இடமும் இல்லாமல் லக்னத்தை சரியாக கணக்கிட இயலாது. {% else %} The Lagna is the zodiac sign rising on the eastern horizon at your exact birth time and place. It represents your physical body, personality, and overall life direction. This is why accurate birth time and place are crucial — the Lagna changes every 2 hours! {% endif %}
{% if language=='ta' %} Vimshottari Dasha என்பது 120 ஆண்டுகள் கொண்ட ஒரு கிரக காலவரிசை. ஒவ்வொரு கிரகமும் குறிப்பிட்ட ஆண்டுகளுக்கு உங்கள் வாழ்க்கையை நிர்வகிக்கிறது. Ketu-7 ஆண்டுகள், Venus-20 ஆண்டுகள், Sun-6 ஆண்டுகள், Moon-10 ஆண்டுகள் என்று மாறுகிறது. {% else %} Vimshottari Dasha is a 120-year planetary period system. Each planet rules your life for a specific number of years: Ketu 7 yrs → Venus 20 yrs → Sun 6 yrs → Moon 10 yrs → Mars 7 yrs → Rahu 18 yrs → Jupiter 16 yrs → Saturn 19 yrs → Mercury 17 yrs. Your birth Nakshatra determines which Dasha you start in. {% endif %}

Geographic coordinates மூலம் சரியான Lagna கணக்கிடப்படுகிறது. சாதாரண பயன்பாட்டிற்கு இது மிகவும் சரியானது. {% else %} AstroGuy AI uses the Lahiri Ayanamsa (the standard for Vedic astrology) and the ephem astronomy library for precise planetary positions. Geographic coordinates are resolved via Nominatim for accurate Lagna calculation. For major life decisions, always consult a qualified human astrologer. {% endif %}

⭐ {% if language=='ta' %}வேத ஜோதிட அடிப்படை{% else %}Vedic Astrology Basics{% endif %}

{% if language=='ta' %} நட்சத்திரங்கள் 360° சுற்றை 27 சம பகுதிகளாக பிரிக்கின்றன (ஒவ்வொன்றும் 13°20'). சந்திரன் எந்த நட்சத்திரத்தில் இருக்கிறதோ அதுவே உங்கள் நட்சத்திரம். இது உங்கள் ஆழமான ஆளுமை, அதிர்ஷ்டம் மற்றும் தசை வரிசையை தீர்மானிக்கிறது. {% else %} The 27 Nakshatras divide the 360° zodiac into 27 equal segments of 13°20' each. The Nakshatra occupied by your Moon at birth is your birth star. It determines your deeper personality traits, compatibility, and Dasha sequence. Each Nakshatra has 4 Padas (quarters) making 108 total divisions. {% endif %}
{% if language=='ta' %} Ayanamsa என்பது வெப்பமண்டல (tropical) மற்றும் நாட்சத்திர (sidereal) ராசிக்கட்டங்களுக்கு இடையேயான வித்தியாசம். Vedic ஜோதிடம் Lahiri ayanamsa (~24°) பயன்படுத்துகிறது, இது Earth's precession (அயன சலனம்) ஆல் ஏற்படுகிறது. {% else %} Ayanamsa is the angular difference between the tropical zodiac (used in Western astrology) and the sidereal zodiac (used in Vedic astrology). AstroGuy uses Lahiri Ayanamsa (~24° currently), which accounts for Earth's precession of the equinoxes over thousands of years. {% endif %}

💑 {% if language=='ta' %}திருமண பொருத்தம்{% else %}Marriage Compatibility{% endif %}

{% if language=='ta' %} Rajju பொருத்தம் Vedic திருமண பொருத்தத்தில் மிக முக்கியமான ஒன்று. இது கணவனின் ஆயுளுடன் தொடர்புடையது. Rajju தோஷம் இருந்தால், பரிகாரங்கள் செய்த பிறகே திருமணம் செய்துகொள்வது நல்லது. {% else %} Rajju Porutham is considered the most critical of all poruthams in South Indian Vedic tradition. The 5 Rajju groups (Sirasu, Kanta, Nabhi, Kati, Pada) each affect different aspects — the most serious being Sirasu Rajju which affects the husband's longevity. Knowledgeable astrologers can identify remedies to mitigate this dosha. {% endif %}
{% if language=='ta' %} நாடி தோஷம் என்பது இருவரும் ஒரே நாடியில் (Vata, Pitta, அல்லது Kapha) பிறந்திருக்கும்போது ஏற்படுகிறது. இது குழந்தைகளின் ஆரோக்கியத்தை பாதிக்கலாம். Nadi Nivarana Puja செய்வதன் மூலம் இந்த தோஷத்தை குறைக்கலாம். {% else %} Nadi Dosha occurs when both partners share the same Nadi (Vata/Pitta/Kapha — based on their birth Nakshatra). It indicates potential health issues for children and genetic incompatibility. The Nadi Nivarana Puja and specific charitable acts are recommended as remedies. {% endif %}
